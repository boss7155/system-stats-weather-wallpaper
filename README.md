## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🌡️ **Виджет погоды** | Реальное время, температура, описание — данные с OpenWeatherMap |
| 🕐 **Часы и дата** | Аналоговые часы с автоопределением часового пояса |
| 📊 **Мониторинг системы** | Живые графики CPU, GPU, RAM и сети в реальном времени |
| 🎵 **Аудиовизуализатор** | Красивый эквалайзер, который появляется только при воспроизведении музыки |

| 🌙 **Ночной режим** | Автоматическое переключение на ночной фон с 22:30 до 6:00 |
| 🎨 **Плавные переходы** | Кроссфейд между фонами при смене погоды |

## 🚀 Установка

### Способ 1: ZIP-архив (Рекомендуется)

> ⚠️ **Важно!** Полная функциональность (мониторинг системы, музыка) работает **только при загрузке через ZIP/папку**. При загрузке через URL графики работают в режиме симуляции.

1. Скачайте последнюю версию:
   - Нажмите **Code → Download ZIP** на странице репозитория
   - Или [скачайте напрямую](https://github.com/boss7155/system-stats-weather-wallpaper/archive/refs/heads/main.zip)

2. Откройте **Lively Wallpaper**

3. Перетащите скачанный ZIP-файл прямо в окно Lively

4. Готово! Обои появятся в вашей коллекции

### Способ 2: Через URL (Ограниченная функциональность)

1. Откройте **Lively Wallpaper**
2. Нажмите **Add Wallpaper** → **Open URL**
3. Вставьте ссылку:
   ```
   https://boss7155.github.io/system-stats-weather-wallpaper/
   ```
4. Нажмите **OK**

> 📝 **Примечание:** При загрузке через URL графики будут отображать симулированные данные, а аудиовизуализатор и информация о треке не будут работать. Для полной функциональности используйте Способ 1.

## ⚙️ Настройка

### Город и API ключ погоды

По умолчанию обои показывают погоду для **Бреста, Беларусь**. Чтобы изменить город:

1. Откройте файл `js/script.js`
2. Найдите блок `CONFIG` в начале файла:
   ```javascript
   var CONFIG = {
       weatherApiKey: 'YOUR_OPENWEATHERMAP_API_KEY',
       weatherCity: 'Brest',           // ← Ваш город
       weatherCountryCode: 'BY',       // ← Код страны
       timezone: 'Europe/Minsk',       // ← Часовой пояс
       weatherLang: 'ru',              // ← Язык прогноза
       weatherRefreshInterval: 10 * 60 * 1000,  // Обновление каждые 10 мин
       weatherUnits: 'metric',         // metric = °C, imperial = °F
   };
   ```

3. Измените значения:
   - `weatherCity` — название города на английском (например: `Moscow`, `London`, `Tokyo`)
   - `weatherCountryCode` — код страны ISO 3166-1 alpha-2 (например: `RU`, `GB`, `JP`)
   - `timezone` — часовой пояс IANA (например: `Europe/Moscow`, `America/New_York`)
   - `weatherUnits` — `metric` для °C или `imperial` для °F
   - `weatherLang` — язык описания погоды (`ru`, `en`, `de`, `fr` и др.)

4. Для своего API ключа (опционально):
   - Зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/api)
   - Получите бесплатный API ключ
   - Замените значение `weatherApiKey`

### Ночной режим

Ночной фон автоматически включается с **22:30 до 6:00** по настроенному часовому поясу. Время нельзя изменить через конфиг — для этого отредактируйте функцию `isNightTime()` в `js/script.js`.

## 🎵 Аудиовизуализатор

Эквалайзер отображается **только когда играет музыка** и автоматически скрывается при паузе или остановке воспроизведения.

### Поддерживаемые плееры

| Плеер | Работает? | Примечание |
|-------|-----------|------------|
| **Spotify** | ✅ Да | Полная поддержка |
| **Браузер** (Chrome, Edge, Firefox) | ✅ Да | YouTube, Яндекс Музыка и др. |
| **VLC** | ✅ Да | Если включён SMTC |
| **AIMP** | ✅ Да | С плагином [Windows 10 Media Control](https://aimp.ru/?do=catalog&rec_id=1097) |
| **foobar2000** | ❌ Нет | Не поддерживает Windows SMTC |

> 💡 **Совет:** Lively Wallpaper использует Windows **System Media Transport Controls (SMTC)** для получения информации о воспроизведении. Плееры, не интегрированные с SMTC, не будут отображаться в визуализаторе.

## 🖼️ Погодные фоны

Обои автоматически меняют фон в зависимости от текущей погоды:

| Фон | Условие |
|-----|---------|
| ☀️ `sunny.jpg` | Ясно, переменная облачность |
| 🌙 `night.jpg` | Ночь (22:30–6:00) или ясная ночь |
| ☁️ `cloudy.jpg` | Облачно, туман |
| 🌧️ `rain.jpg` | Дождь, морось |
| ❄️ `snow.jpg` | Снег |
| ⛈️ `storm.jpg` | Гроза |

Фоны находятся в папке `textures/`. Вы можете заменить их на свои изображения с теми же названиями.

## 🔧 Технологии

- **[Lively Wallpaper](https://github.com/rocksdanister/lively)** — движок для динамических обоев Windows
- **[Chart.js](https://www.chartjs.org/)** + **chartjs-plugin-streaming** — графики в реальном времени
- **[OpenWeatherMap API](https://openweathermap.org/api)** — данные о погоде
- **CSS-анимации** — анимированные иконки погоды (солнце, луна, облака, дождь, снег, молния)
- **Windows SMTC** — интеграция с медиаплеерами через System Media Transport Controls
